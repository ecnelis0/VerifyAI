How it works

VerifyAI addresses training data poisoning by using consensus — where models disagree, you know to be cautious.

1

Sensitive word detection

Your question is scanned for keywords related to medicine, toxicity, self-harm, and safety before anything is sent to an AI.

2

Three parallel queries

The question is sent simultaneously to three AI models, each with a different system prompt — cautious, factual, and safety-focused.

3

Intersection analysis

A fourth call compares all three responses, finds agreement points, flags contradictions, and scores confidence per model.

4

Trust score & verdict

You receive a reliability label (High / Medium / Low), a confidence percentage, and a verified summary you can actually trust.

This is a static HTML file. Replace YOUR_API_KEY in the JS with your own Anthropic API key, or proxy the request through your backend for security.

// ── Core verification function ──
// Drop this into your own JS or Node backend

async function verifyQuestion(question) {
  const API_KEY = "YOUR_API_KEY";  // ← replace or proxy
  const ENDPOINT = "https://api.anthropic.com/v1/messages";

  // Three different model personas
  const personas = [
    "You are a cautious medical/safety AI. Answer in 2-3 sentences, warn of risks.",
    "You are a precise factual AI. Answer in 2-3 sentences with accuracy and safety notes.",
    "You are a safety-first AI. Answer in 2-3 sentences, prioritize user safety."
  ];

  // Fire all three in parallel
  const responses = await Promise.all(
    personas.map(sys =>
      fetch(ENDPOINT, {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          "x-api-key": API_KEY,        // ← add this for direct calls
          "anthropic-version": "2023-06-01"
        },
        body: JSON.stringify({
          model: "claude-sonnet-4-20250514",
          max_tokens: 300,
          system: sys,
          messages: [{ role: "user", content: question }]
        })
      }).then(r => r.json())
        .then(d => d.content[0].text)
    )
  );

  // Synthesize into a trust report
  const synthPrompt = `Compare these three AI responses and return JSON:
Model A: ${responses[0]}
Model B: ${responses[1]}
Model C: ${responses[2]}
Return: { overallConfidence, modelConfidences[],
          agreementPoints[], disagreements[],
          verdict, reliabilityLabel }`;

  const synthesis = await fetch(ENDPOINT, { /* same headers */ });
  return JSON.parse(await synthesis.json());
}
